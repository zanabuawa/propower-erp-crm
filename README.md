# ERP Empresarial para Electroconstructoras

Este proyecto nace de una necesidad real dentro de una empresa contratista del sector eléctrico. Su objetivo es centralizar en una sola plataforma los procesos administrativos, comerciales y operativos que normalmente se gestionan mediante hojas de cálculo, documentos dispersos y múltiples sistemas independientes.

El ERP ha sido desarrollado desde cero utilizando Laravel 12, Livewire 3 y MySQL, con un enfoque modular que permite adaptar la plataforma a diferentes áreas de la empresa y facilitar su crecimiento a futuro.

Actualmente el sistema integra más de 12 módulos orientados a la gestión diaria de la organización.

## Funcionalidades principales

- Gestión de clientes y seguimiento comercial (CRM)
- Cotizaciones y ventas
- Compras y requisiciones
- Administración de proveedores
- Inventario multi-almacén
- Transferencias entre sucursales
- Gestión de productos y existencias
- Recursos Humanos y reclutamiento
- Control de asistencia
- Roles y permisos
- Dashboard con indicadores operativos
- Notificaciones automatizadas
- Facturación CFDI 4.0 mediante FacturAPI
- Soporte para múltiples empresas dentro de una misma instalación

## Tecnologías utilizadas

### Backend
- PHP 8.3
- Laravel 12
- Livewire 3

### Frontend
- Tailwind CSS
- Alpine.js
- Chart.js

### Base de datos
- MySQL

### Integraciones
- FacturAPI
- SendGrid

## Filosofía del proyecto

Más que un sistema administrativo tradicional, este ERP busca convertirse en una herramienta que permita a la empresa tener control y trazabilidad de sus operaciones, reduciendo tareas manuales y facilitando la toma de decisiones mediante información centralizada y actualizada.

El diseño modular permite incorporar nuevas áreas de negocio conforme la empresa lo requiera, manteniendo una arquitectura organizada y escalable.

## Estado del proyecto

Proyecto en desarrollo activo.

Actualmente se continúa trabajando en la optimización de procesos existentes y en la incorporación de nuevos módulos orientados a la gestión de proyectos, ingeniería y control operativo.
